Image Card Grid

A single, reusable WordPress block: a configurable grid or carousel of image cards with optional text. Every option below is built in, so any layout is possible from the editor. The partner award badges are one preset of it. Visual design: Figma file.

Options (block attributes)

Display mode

  • Mode: grid or carousel
  • Carousel: arrows, dots, items visible, edge fade, autoplay

Grid layout

  • Columns (desktop): 2–6
  • Responsive: auto step-down, or per-breakpoint
  • Gap between cards
  • Alignment: left / centred

Image

  • Size: small / medium / large
  • Shape: circle / square / rounded
  • Fit: contain / cover

Card

  • Chrome: bare image or card
  • Border (colour, width), corner radius, padding
  • Background, shadow, hover lift
  • Clickable: link whole card to a URL

Card text (optional)

  • Show text on/off
  • Title (1–2 lines), subtitle, meta line
  • Alignment

Block container

  • Padding, background, border + radius
  • Width: contained / full

Above the grid (optional)

  • Heading / overline
  • Intro text or written list
  • Divider

Card data (repeater)

  • Each card: image (required), title, subtitle, meta, link

Behaviour notes

  • Cards are a repeatable list; the block renders however many are added.
  • Grid and carousel share the same card; switching mode does not change the data.
  • Should handle 1 to ~24 cards gracefully.
  • "Card chrome off" = bare images (the badge use case); "on" = bordered cards.
