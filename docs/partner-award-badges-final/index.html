Awards block for Partner Finder

The chosen approach, plus the two variations to build so we can compare them on the page.

✓ Chosen approach

One badge per award, logo kept

Show each award once, with the partner logo on the medallion so it stays bespoke. When a partner wins the same award across several years, we never repeat the badge. The year is shown one of two ways below. Please build both.

Variation A — Year in the badge

The medallion carries the year(s). Badges sit in a simple row or grid.

Badge carries the year

Left: two years. Right: four years. Watch-out: the more years, the tighter the badge gets, so the layout should accommodate a range of badge sizes.

Variation B — Year as text

The badge can be yearless. The year(s) sit in a text callout beneath each one, which scales cleanly no matter how many years.
